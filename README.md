Paper-Style Tic Tac Toe

This is a web-based, fully responsive Tic Tac Toe game with a unique "paper-style" aesthetic. Built from scratch with vanilla HTML, CSS (using TailwindCSS), and JavaScript, it mimics the classic pen-and-paper feel by only showing the inner grid lines.

‚ú® Features

Unique Paper-Style Grid: The game board only shows the inner lines, just like a hand-drawn grid.

Custom Player Names: Start the game by entering names for Player X and Player O.

Dynamic Demo Animation: While entering names, a random demo animation plays on the game board in the background.

Win/Draw Detection: The game automatically detects all win conditions (horizontal, vertical, diagonal) and draw states.

Celebration Animations:

Winning Line: A line strikes through the three winning cells.

Confetti Effect: A fun party-popper confetti animation plays on a win.

Fully Responsive: The game looks and plays great on both desktop and mobile devices.

Play Again: A "Dobara Khelain" (Play Again) button allows for quick rematches.
